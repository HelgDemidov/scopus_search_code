# Запрет на проактивное создание файлов и папок

Никогда не создавай новые файлы или папки (включая конфигурационные файлы, скрытые файлы типа .agyignore и рабочие директории) по собственной инициативе. 

Ты должен всегда ждать прямой, недвусмысленной команды от пользователя (например, "создай папку X", "запиши это в файл Y"), прежде чем использовать инструменты записи или создания директорий. Если у тебя есть инициатива создать файл для удобства — сначала спроси пользователя.
