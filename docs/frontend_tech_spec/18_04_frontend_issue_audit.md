```json

  "total_articles": 6782,
  "total_journals": 2169,
  "total_countries": 94,
  "open_access_count": 2892,
  "by_doc_type": [
    { "label": "Article", "count": 4377 },
    { "label": "Conference Paper", "count": 1396 },
    ...
  ]
}
```

Данные полностью совпадают с отображением на `/explore`. Страница доступна
незарегистрированному пользователю и показывает кураторскую коллекцию из 6 782 статей.

Цепочка исправлений, приведших к результату:
1. **`ac36869`** — устранён дублирующий `Depends(get_scopus_client)` → `RuntimeError: client is closed → 500`

3. **`aefe328`** — `save_many` возвращает объекты с `id` из БД → устранён `ValidationError: id=None`
